# Product Overview

IWork（爱工作）是一款面向开发者的桌面工具集应用，基于 Electron + Vue 3 + TypeScript 构建。

## 核心功能

### 1. 截图工具
- 快捷截图（Ctrl+Shift+X）
- 区域选择 + 标注工具（箭头、矩形、文字、马赛克）
- 截图历史自动保存
- 置顶显示功能

### 2. SSH 客户端
- 连接管理（保存常用服务器配置）
- 密码和密钥认证
- xterm.js 终端（支持 vim、nano 等交互式应用）
- 快捷命令模板
- 文件浏览和上传下载（SFTP）

### 3. 剪贴板历史
- 自动监控和保存剪贴板内容
- 模糊搜索历史记录
- 支持文本和图片

### 4. 知识库管理
- Markdown 编辑和实时预览
- 分类和标签管理
- 全文搜索

### 5. 开发工具集（40+）
- 编解码：Base64、URL、Unicode、HTML
- 格式化：JSON、SQL、XML/YAML
- 加密：MD5、SHA、AES、DES、JWT
- 生成器：UUID、密码、二维码
- 网络：HTTP 客户端、端口扫描、WebSocket
- 其他：时间戳转换、正则测试、文本对比、Cron 表达式

## 目标用户

需要日常开发辅助工具的程序员和运维人员。

## 技术特点

- 纯本地应用，数据私有（保存在 appData 目录）
- 跨平台（Windows/macOS/Linux）
- 深色主题，科技感界面
- 快捷键支持
